# QSDemo - 專業問卷系統前端專案

## 專案概述

QSDemo 是一個基於 Angular 19 打造的高級問卷管理系統。本專案採用專業的 **Glassmorphism (毛玻璃)** 設計風格與溫暖的漸層配色，提供流暢的使用者填寫體驗與功能強大的管理後台。

## 線上展示

🌐 **GitHub Pages**: [https://AtayalLin.github.io/Angular-qsdemo/](https://AtayalLin.github.io/Angular-qsdemo/)

## 核心功能亮點

### 1. 智慧問卷列表
- **進階搜尋**：支援名稱模糊比對、**日期區間選擇**、類型與狀態篩選。
- **動態分頁**：實作 10 頁視窗切換邏輯，支援大量數據瀏覽。
- **身分識別**：區分訪客、一般會員與管理員，根據權限動態開放功能。
- **安全刪除**：自定義防呆確認彈窗，避免誤刪風險。

### 2. 專業管理中心 (Admin Control Panel)
- **卡片式編輯器**：每一題均為獨立大卡片，層次分明，編輯體驗極佳。
- **問卷設定**：支援名稱、類型、說明及開放時間的結構化配置。
- **動態題目管理**：
  - **基本資料勾選**：自由決定是否收集姓名、電話、信箱。
  - **題型支援**：單選（含邏輯跳題預留）、多選（含最高項數限制）、簡答（含字數限制）。
- **工作台模式**：底部設有「待處理問卷」區塊，方便快速載入草稿進行編輯。

### 3. 會員系統
- **註冊與登入**：完整的會員註冊流程，資料儲存於 LocalStorage。
- **權限區分**：登入會員未來可查詢填答歷史；管理員可進入後台。

## 技術架構

- **框架**: Angular 19 (Standalone Components)
- **語言**: TypeScript 5.6+
- **樣式**: SCSS (CSS Grid & Flexbox 佈局)
- **視覺**: Glassmorphism 毛玻璃效果、自定義漸層系統
- **圖表**: Chart.js 統計視覺化

## 開發與部署

### 快速開始
1. `npm install` - 安裝依賴。
2. `npm start` - 本地開發環境。

### 自動化部署
本專案已配置部署腳本，執行以下指令即可生成支援 SPA 路由（404 修復）的編譯版本：
```bash
npm run deploy
```
