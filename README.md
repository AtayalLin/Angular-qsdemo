# QSDemo - 問卷系統前端專案

## 專案概述

QSDemo 是一個基於 Angular 19 的問卷系統前端應用程式，提供完整的問卷管理、填寫和統計功能。系統支援管理者模式和使用者模式，包含問卷列表、填寫、預覽、結果統計以及使用者註冊等功能。

## 主要功能

### 1. 問卷列表頁面 (`/surveys`)

- **功能描述**: 顯示所有問卷的列表，支援關鍵字、日期區間、類型與狀態搜尋。
- **管理者模式**:
  - 查看所有問卷狀態（已發佈、已儲存尚未發佈、草稿）。
  - 發佈、編輯、刪除問卷。
  - **管理中心入口**: 登入後可見綠色「管理中心」按鈕。
- **分頁功能**:
  - 列表底部支援數位分頁。
  - **動態頁籤**: 以 10 頁為一個視窗動態顯示。
  - **空頁提示**: 當切換至無資料的頁碼時，顯示「此頁無問卷」提示。

### 2. 管理中心 (`/admin`) [NEW]

- **功能描述**: 專屬管理者的集中管理介面，採用專業的毛玻璃風格與卡片排版。
- **管理頁籤**:
  - **問卷設定**: 編輯問卷名稱、說明及有效期間。
  - **題目編輯**: 自定義問卷題目（單選、多選、文字）。
  - **問卷回饋**: 查看使用者填寫的詳細數據（待串接）。
  - **統計分析**: 視覺化圖表分析（待串接）。

### 3. 問卷填寫頁面 (`/surveys/:id/question`)

- **功能描述**: 使用者填寫問卷的介面，支援單選、多選、文字題。

### 4. 註冊頁面 (`/register`)

- **功能描述**: 提供使用者註冊功能，目前是資料儲存於 LocalStorage，具備密碼一致性與帳號重複檢查。未來會使用MySQL儲存個人資料

## 技術架構

- **Angular 19**: Standalone Components 架構
- **TypeScript**: 強型別開發
- **SCSS**: 自定義溫暖色系主題、毛玻璃特效（Glassmorphism）
- **Chart.js**: 統計圖表實作

## 開發與部署

### 安裝步驟

```bash
npm install
```

### 本地開發

```bash
npm start
```

### 部署至 GitHub Pages

本專案已設定好部署腳本，執行以下指令後，將生成的 `dist/qsdemo/browser` 內容上傳至 `gh-pages` 分支即可：

```bash
npm run deploy
```

> **腳本功用**: 自動設定 `base-href` 為 `/Angular-qsdemo/` 並生成 `404.html` 以支援 SPA 路由。
